# Jogo da Velha  

Projeto desenvolvido durante as aulas de **Introdução à Programação Front-end e Back-end**, da trilha **TEC Desenvolvimento de Software do SCTec**.

Aplicação web do clássico Jogo da Velha construída com HTML, CSS e JavaScript puro, com foco na consolidação de lógica de programação e manipulação do DOM.

## Funcionalidades

- Alternância automática entre jogador X e O  
- Bloqueio de jogadas em células já selecionadas  
- Verificação automática de vitória  
- Verificação de empate  
- Reinício completo da partida  

## Tecnologias Utilizadas

- HTML5  
- CSS3 (Grid Layout)  
- JavaScript (ES6+)  

## Competências Demonstradas

- Manipulação e atualização dinâmica do DOM  
- Tratamento de eventos com `addEventListener`  
- Estruturação de lógica condicional  
- Uso de arrays, métodos `some()` e `every()`  
- Controle de estado da aplicação  

## Objetivo do Projeto

Aplicar conceitos fundamentais de desenvolvimento web em um projeto funcional, reforçando organização de código, lógica estruturada e boas práticas básicas de JavaScript.
